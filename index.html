前端UI框架

前端UI框架大荟萃，盘点那些你熟悉的， 陌生的，响应式的， mobile first的UI Framework。

-by 6174



本人有UI 框架收集癖， 恰好最近也在做一个UI框架， 收集也顺带学习，这里整理出我所喜欢的前端UI框架（对的， 不是全部， 有很多比较老的框架我就不再整理了，因为我只想整理实用的干货） 在列举的同时会介绍相应框架的优缺点。 以下排名不分先后。 大家有什么其他的优秀框架推荐， 也可以整理在这里， 你只需要到这里打一个patch－－－> github/ui-framework

#1.

Bootstrap v3

less ie7+ mobile first

官方网站：http://www.bootstrap.com

中文官网：http://www.bootcss.com

官方介绍： Bootstrap 是最受欢迎的 HTML、CSS 和 JS 框架，用于开发响应式布局、移动设备优先的 WEB 项目。为所有开发者、所有应用场景而设计。 Bootstrap 让前端开发更快速、简单。所有开发者都能快速上手、所有设备都可以适配、所有项目都适用。

对于中文开发者来说， bootcss是一个不错的学习网站， 在bootcss中可以看到很多其他针对bts的工具，插件和使用案例

#2.

Foundation

sass ie9+ responsive

官方网站：http://www.foundation.com

官方介绍： The most advanced responsive front-end framework in the world

Foundation 相比bts拥有更加全面的网格系统设计，为响应式而生， 同时拥有丰富的组件，foudation和bts在前端UI框架中都拥有相当高的人气，相信对于大多只了解过bts的同学来说，学习一下foudation会有很大收获。 “Foundation是第一款专门帮助用户创建能够运行在任何设备上的响应式站点的CSS框架，”Web设计机构Zurb合作伙伴兼Foundation设计主管Jonathan Smiley指出。“我们自发布首个公共版本开始就一直致力于打造响应式站点，而且在经历了五个版本的实践研发过程后、我们已经亲眼见证了众多开发者在创建多设备应用及站点方案时所遭遇的难题，因此有能力帮助他们顺利度过难关。”

#3.

Semantic-UI

less semantic responsive

官方网站：http://semantic-ui.com/

官方介绍： Semantic empowers designers and developers by creating a language for sharing UI.

目前的版本为0.19.0，个人觉得最有潜力的UI框架， 语义化的设计使得UI的编辑极富表现力，丰富而且精致的UI组件是其他UI框架不能比的， 同时UI组件库还在不断的扩充. 官网的介绍说在组件的调试方面是他们的优点， 可以看到组件运行的life cycle log信息。 但是对于我来说， 我不想说这是优点， 源码太蛋疼， 极其长， 因为加了很多模块调试的代码。

#4.

gumbyframework

sass responsive customlize

官方网站：http://gumbyframework.com/

官方介绍： Gumby 2 is an amazing responsive CSS Framework. It is Flexible and Insanely Simple

Gumby官网的设计非常漂亮， 大背景图片，恰当的padding, margin, 客人的按钮。 不信戳进去看就知道了， gumby里边有很多有趣的js组件，如paralax, fittext, responsive image, fix.

#5.

Base

sass + less responsive

官方网站：http://matthewhartman.github.io/base

介绍： Base是一款轻量级CSS预处理工具，能够将代码编译为SASS(即Syntactically Awesome CSS)或者LESS(Leaner CSS)。尽管已经有很多框架放弃了IE浏览器的支持，Base仍然坚定扮演着IE拥护者的角色，Matthew Hartman表示——他设计Base的初衷在于帮助设计师们编写出更为简洁且结构清晰的CSS代码。

#6.

Kube

官方网站：http://imperavi.com/kube

介绍： Kube标榜自身是一款“专业级”CSS框架。尽管其它框架同样能够提供预定义样式以构建起视觉效果相似的站点外观，但Kube的使用感受更加简单，Kube项目首席开发者之一Artem Rosnovskiy解释称。

#7.

Ratchet

mobile－only sass

官方网站：http://goratchet.com/

介绍： Ratchet 是bootstrap group针对mobile的UI框架，所以我就不用怎么夸它了， 到官网上看看就知道它的是bootstrap的兄弟， ratchet的css样式是针对ios的， 如果想自定义自己的皮肤，还得手动改sass, 对的, 不是less。

#8.

Pure

mobile css only

官方网站：http://purecss.io/

官方介绍： Pure is ridiculously tiny. The entire set of modules clocks in at 4.4KB* minified and gzipped. Crafted with mobile devices in mind, it was important to us to keep our file sizes small, and every line of CSS was carefully considered. If you decide to only use a subset of these modules, you'll save even more bytes.

看看它的官网你就会喜欢上它， 简介，小巧，only css. 官网有丰富的案例

#9.

Webix

mobile

官方网站：http://webix.com/

介绍： webix不同于以上的任何一个UI框架， webix的哲学是通过简介的js代码创建丰富的UI， 看看官网首页就知道了， 开发人员完全不用考虑布局css， 只有考虑数据。 对于自定义皮肤还提供一个skin builder工具。 webix拥有极其丰富的组件库，以及整合了angular和backbone。 对于不太了解前端的开发人员， 这应该是个不错的选择。

#10.

Framework7

mobile－only

官方网站：http://goratchet.com/

官方介绍： Full Featured HTML Framework For Building iOS Apps

正如其官介，功能全面的针对ios的web app ui框架， 对于想开发ios web app的同学， 可以对比ratchet一起参考取长补短。

#11.

ionic

mobile－only sass native

官方网站：http://ionicframework.com/

官方介绍： Create hybrid mobile apps with the web technologies you love.

整合angular， 和上面介绍个的针对mobile的开发不同的是，ionic支持native（phoneGap）， 可以用它创建ios或者android应用。

#12.

kendo-ui

mobile

官方网站：http://www.telerik.com/kendo-ui

官方介绍： Everything you need to build sites and apps with pure JavaScript and HTML5

70+ Kendo UI Widgets and Framework，支持付费和免费版本。

#13.

JqueryMobile

mobile

官方网站：http://jquerymobile.com/

官方介绍：jQuery Mobile is a HTML5-based user interface system designed to make responsive web sites and apps that are accessible on all smartphone, tablet and desktop devices.

除了比较丑以外（皮肤可以自己搞），配合backbone还是可以做些东西

#14.

sencha Touch

mobile-only

官方网站：http://www.sencha.com/

官方介绍： With over 50 built-in components, themes for every popular mobile platform, and a built-in MVC system, Sencha Touch provides everything you need to create powerful, universal mobile web apps.

#15.

onsenui

mobile-only native responsive

官方网站：http://onsenui.io/

官方介绍：The Answer to PhoneGap UI Development A Custom Elements-Based HTML5 UI Framework, Onsen UI is a perfect solution for developing hybrid apps. That's because our priority is performance and ease of use. Onsen UI is fully open source, and never locks you in. It also provides the option of deploying your apps to the server to use as a tool to build a performance website.

官方网站：http://concisecss.com/

官方介绍：Lean, mean and clean
